# Kuma Mobile
Kuma Books is a forum that unites book readers and exchange their opinions about books that have been read

## Installation

Clone this repo

```bash
https://github.com/kubil-ismail/kuma_mobile.git
```
Intall dependency.

```bash
cd kuma_mobile
npm install
npm run android
```
